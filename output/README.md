# Output Directory

This directory contains generated reports and output files from the Anypoint Platform assessment process.

## Files

### Platform Assessment Reports
- **anypoint-platform-assessment-report-YYYYMMDD-HHMMSS.md**: Generated reports containing the hierarchical structure of the Anypoint Platform organization, including business groups, environments, and their relationships.

## File Naming Convention

Reports follow the naming pattern:
```
anypoint-platform-hierarchy-report-[DATE]-[TIME].md
```

Where:
- `DATE`: YYYYMMDD format (e.g., 20260213)
- `TIME`: HHMMSS format (e.g., 113332)

## Usage

These output files are automatically generated by the assessment workflow and provide detailed insights into the current Anypoint Platform configuration. They can be used for:

- Organization structure analysis
- Environment mapping
- Governance and compliance reviews
- Migration planning
- API-Led Connectivity assessments

## Archive Policy

Consider archiving older reports periodically to maintain organization. Reports are timestamped for easy identification and chronological sorting.
